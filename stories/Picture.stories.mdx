import { Meta, Story, Canvas } from '@storybook/addon-docs/blocks'
import avatar from './assets/guilherme.png'
import Picture from './Picture'

<Meta title="Documentation/Content/Picture" component={Picture} />

A Picture is a component used to display a picture. It comes in two default sizes,
`large` and `small`, though you can also give it a specific pixel size.

<Canvas>
  <Picture source={avatar} size="300" />
  <Picture source={avatar} size="large" />
  <Picture source={avatar} size="small" />
</Canvas>

It also gracefully handles missing pictures

<Canvas>
  <Picture source={null} size="300" />
  <Picture source={null} size="large" />
  <Picture source={null} size="small" />
</Canvas>
